# My .dotfile config for nvim

This is my configuration for neovim. Currently I am using version 0.11.3 with [catpuccin-macchiato](https://github.com/catppuccin/nvim) colorscheme. 

## Plugins

This is a breakdown of the plugins I have installed using [Lazy](https://github.com/folke/lazy.nvim) as plugin manager):

- Some autocompletion plugins (in ```autocompletion.lua```)
- ```git``` integrations (in ```gitFunctionality.lua```)
- LSP configuration (in ```lspconfig.lua```)
- [bufferline](https://github.com/akinsho/bufferline.nvim) create a nice bufferline listing all open buffers
- [dashboard](https://github.com/folke/snacks.nvim) Plugin to create a dashboard when opening nvim
- [guess-indent](https://github.com/NMAC427/guess-indent.nvim) A smart indenter
- [indent-blankline](https://github.com/lukas-reineke/indent-blankline.nvim) A visual guide for indentation levels
- [luasnip](https://github.com/L3MON4D3/LuaSnip) Snippet engine
- [mini](https://github.com/echasnovski/mini.nvim) Suite of small plugins
- [neotree](https://github.com/nvim-neo-tree/neo-tree.nvim) Nicer file navigation
- [telescope](https://github.com/nvim-telescope/telescope.nvim) Fuzzy finder
- [treesitter](https://github.com/nvim-treesitter/nvim-treesitter) Parser for syntax trees
- [vimtex](https://github.com/lervag/vimtex) general LaTeX support 
- [which-key](https://github.com/folke/which-key.nvim) shows pending commands
